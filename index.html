GO

三行装好 · 没什么玄学

# 1. 拉仓库 + 装依赖（需要装 uv）
git clone https://github.com/Tsai6688/HoldSayByDouBao.git
cd  HoldSayByDouBao && uv sync

# 2. 填入你自己的豆包凭证（volcengine.com 新号有免费额度）
cp .env.example .env   # 然后编辑 .env 填 APP_ID / TOKEN

# 3. 安装 LaunchAgent 并启动
chmod +x doubaoctl
./doubaoctl install
./doubaoctl grant     # 打开系统设置协助授权（辅助功能 / 输入监控）
./doubaoctl start

# 完事。按住 F5 试试。
